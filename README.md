# Financial assistant

## Description

AI Financial Assistant — прототип интеллектуального ассистента для частного инвестора.
Его задача — помогать анализировать портфель, понимать влияние новостей на активы и проверять инвестиционные идеи с использованием актуальных рыночных данных.

### Problem Definition

Сейчас частный инвестор сталкивается с несколькими проблемами:
- информация о компаниях и рынках разбросана по разным источникам (новости, аналитика, отчёты);
- сложно быстро понять, какие новости реально влияют на портфель;
- проверка инвестиционных идей требует ручного поиска данных и анализа;
- многие AI-ассистенты дают общие ответы без связи с реальными данными портфеля.

Проект предлагает другой подход: ассистент работает с конкретным портфелем пользователя, получает данные о компаниях и котировках, а также может использовать интернет-поиск для актуальной информации.

В результате пользователь может задавать вопросы на естественном языке и получать обоснованный анализ, основанный на данных.


## What PoC do

PoC демонстрирует базовые возможности AI-ассистента для анализа инвестиционного портфеля на основе рыночных данных и официальных экономических показателей.

На демо пользователь может подключить брокерский аккаунт (в режиме read-only) и задать вопросы о своём портфеле или отдельных активах.

---

### 1. Импорт портфеля из брокера

Ассистент может получить данные из брокерского API:

- список активов в портфеле  
- количество бумаг  
- текущую стоимость позиций  

На демо показывается:

- структура портфеля  
- стоимость позиций  
- распределение по активам  

---

### 2. Динамика цены активов

Для каждого актива ассистент может показать:

- текущую цену  
- историческую динамику  
- изменение цены за выбранный период  

На демо пользователь увидит:

- график изменения цены  
- базовые показатели изменения (рост / падение / волатильность)

---

### 3. Макрофинансовые показатели

Ассистент может собрать и показать ключевые макроэкономические индикаторы из доверенных источников (например, отчётов центральных банков или официальной статистики).

На демо показывается:

- текущие значения показателей  
- динамика изменения  
- графики или краткая сводка

---

### 4. Анализ портфеля на основе стратегии

Пользователь заранее определяет тип своей стратегии с разной степенью подробности:

> Я предпочитаю консервативный стиль

> Вот мои правила диверсификации в зависимости от показателей актива и экономики: ... 

После сохранения этой информации в память, пользователь может задать вопрос по своему портфелю

> Стоит ли сейчас покупать акции компании X?

> Какие компании из отрасли Y стоит приобрести, а от каких отказаться в моем портфеле

Ассистент выполняет базовый анализ на основе:

- исторической динамики цены актива  
- состояния соответствующего сектора (по официальным отчетам и агрегированным данным)  
- макрофинансовых показателей
- предпочитаемой стратегии от пользователя

Ответ представляет собой **аналитическую интерпретацию доступных данных**, а не инвестиционную рекомендацию.

---

### 5. Автоматическая визуализация данных

На демо показывается, что ассистент может автоматически строить простые визуализации:

- графики изменения цены актива  
- динамику макропоказателей  
- диаграммы структуры портфеля

---

## What PoC NOT do

Чтобы ограничить сложность прототипа, ряд возможностей намеренно не входит в PoC.

---

### Нет автоматической торговли

Ассистент не может:

- покупать активы  
- продавать активы  
- управлять портфелем пользователя  

Интеграция с брокером используется **только для чтения данных**.

---

### Нет анализа новостей

PoC **не анализирует новости компаний или рынков**.

Если по компании нет доступных данных в официальных источниках или статистике, ассистент сообщает пользователю, что информация пока не покрывается системой.

---

### Нет сложных финансовых моделей

PoC не реализует:

- фундаментальные модели оценки компаний  
- продвинутые прогнозные модели  
- количественные торговые стратегии  

---

### Нет анализа инвестиционных стратегий

Система пока не поддерживает:

- backtesting стратегий  
- сравнение инвестиционных стратегий  
- автоматический подбор стратегии  

Этот функционал может быть добавлен позже после обсуждения подхода к оценке качества и метрик.

---

### Нет production-уровня инфраструктуры

PoC не решает задачи:

- масштабирования системы  
- production-уровня безопасности  
- соответствия регуляторным требованиям  
- долгосрочного хранения пользовательских данных
